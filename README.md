# Skore

Simple marionette application to change dom based on interaction.

## Installation

You need node v5.x.x and npm as a prerequisite.

```sh
$ npm i
$ npm i nightwatch -g
$ bower i
```

### Run

```sh
$ gulp serve
```

### Test

In one bash:

```sh
$ java -jar selenium-server-standalone-2.53.0.jar 
```

In another bash:

```sh
$ gulp serve
```

And in another bash:

```sh
$ nightwatch start
```
